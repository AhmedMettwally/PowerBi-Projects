# Global Superstore Dashboard

An interactive Power BI dashboard that analyzes global sales performance, returns, customer segments, and regional trends. 
The dashboard enables users to explore business insights through dynamic filters and multiple report pages.

## Features
- Sales, Profit, Delivery Days, and Returned Orders KPIs
- Sales analysis by region, market, category, and customer segment
- Returned orders analysis and return rate tracking
- Top customers and most/least profitable products
- Customer segmentation analysis
- Interactive year slicer and page navigation

## Tools Used
- Power BI
- Power Query
- DAX

## Dashboard Pages

### All Data
- Sales overview and key KPIs
- Regional sales map
- Category, market, and segment analysis
- Top customers and product profitability

### Returned Data
- Returned orders and return rate analysis
- Returns by category, market, and segment
- Returned products and shipping cost analysis

### Segmentation
- Customer segmentation overview
- Orders, quantities, and returns by customer category
- Customer behavior analysis
