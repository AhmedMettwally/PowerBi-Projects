# Supply Chain Sales Dashboard

An interactive Power BI dashboard that analyzes supply chain sales performance, products, customers, and orders. The dashboard provides key business insights through dynamic visualizations and filters.

## Features
- Sales, Profit, and Average Sales KPIs
- Order analysis by year and city
- Product performance (most/least bought and profitable products)
- Customer analysis (top/lowest customers by sales and profit)
- Sales by category and customer segment
- Shipping mode, payment type, and shipping days analysis
- Interactive year slicer and page navigation

## Tools Used
- Power BI
- Power Query
- DAX

## Dashboard Pages
- Orders
- Products
- Customers
